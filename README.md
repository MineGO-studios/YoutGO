# YoutGO — YouTube Downloader

**Version:** 1.1.3 
**Created:** 2025/6/6  
**Author:** MJ  
**Studio:** MineGO Studio

YoutGO is a modern, multi-threaded YouTube downloader for Windows.  
Built for privacy, speed, and ease of use — with robust playlist, batch, and quality controls.

---

## Features

- Download MP3, MP4, M4A, WAV, and more.
- Select video/audio quality (best, 1080p, 720p, audio-only, etc).
- Multi-threaded batch downloads (up to 5 links at once).
- Download entire playlists (all videos, or just first).
- Clean progress bar, log output, and status view.
- Modern, responsive UI (ttkbootstrap).
- Dark theme, robust error handling.
- Owner: MJ, Studio: MineGO Studio

---

## Usage

1. Paste one or more YouTube links (playlist or video) in the box.
2. Choose output format and quality.
3. (Optional) Enable "Download as playlist".
4. Set max concurrent downloads.
5. Click "Download All".

---

## System Requirements

- Windows 10/11, Python 3.10+ (for script mode)
- Standalone EXE available (see build instructions)
- Requires `ffmpeg` (bundled or on PATH for EXE)
